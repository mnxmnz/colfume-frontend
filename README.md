## Getting Started

#### Prerequisites

```
npm install --global yarn
```

#### Installation

1. Clone the Repo

```
git clone https://github.com/mnxmnz/colfume-frontend.git
```

```
cd colfume-frontend
```

2. Install Project Packages

```
yarn
```

3. Run the Project (Dev Mode)

```
yarn dev
```
